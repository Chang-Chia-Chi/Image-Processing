# Discrete Fourier Transform DFT

## 簡介
傅立葉轉換被廣泛利用在各種工程問題上，對影像處理也不例外，很多時候空間域無法處理的問題，在頻率域都可迎刃而解，以下將簡述離散形式的傅立葉轉換(DFT)，轉換後圖像的物理意義，以及一些基本的應用。
